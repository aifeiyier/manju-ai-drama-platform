## 功能说明

故事策划模块是 AI 漫剧创作链路的第二步。接收选题分析的输出，生成完整的剧本、分镜表、角色圣经和场景库。

核心能力：
- **动态镜头生成** -- 根据题材和故事复杂度自动计算镜头数量和参数
- **节奏配置** -- 每种题材有独立的镜头时长、分布、情绪递进参数
- **角色圣经生成** -- 为每个角色生成视觉特征和性格设定
- **场景库生成** -- 根据题材生成默认场景和关键场景

## 动态镜头生成逻辑

### 计算公式

```
基础镜头数 = max(8, floor(平均时长 / 基础镜头时长))
复杂度分数 = min(1, 冲突点数量 * 0.1 + 关键场景数量 * 0.08 + 文本长度加分)
最终镜头数 = floor(基础镜头数 * (1 + 复杂度分数 * 0.5) * 镜头倍率)
```

### 复杂度计算

| 因素 | 计算方式 | 最大贡献 |
|------|---------|---------|
| 冲突点数量 | min(冲突点数, 5) * 0.1 | 0.5 |
| 关键场景数量 | min(场景数, 5) * 0.08 | 0.4 |
| 文本长度 | >500字 +0.2, >200字 +0.1 | 0.2 |

### 四幕分配

最终镜头数按四幕均分，第三幕（高潮）额外获得余数：

| 幕 | 占比 | 情绪起点 |
|----|------|---------|
| 第一幕（开篇） | 25% | 0.0 |
| 第二幕（发展） | 25% | 0.25 |
| 第三幕（高潮） | 25% + 余数 | 0.5 |
| 第四幕（结局） | 25% | 0.75 |

## 节奏配置

每种题材有独立的节奏参数，控制镜头的时长、类型分布和情绪递进。

### 各题材节奏参数

| 题材 | 基础镜头时长(秒) | 镜头倍率 | 特写占比 | 近景占比 | 中景占比 | 全景占比 |
|------|-----------------|---------|---------|---------|---------|---------|
| 玄幻 | 4 | 1.2 | 25% | 30% | 25% | 20% |
| 都市 | 5 | 1.0 | 20% | 35% | 30% | 15% |
| 甜宠 | 6 | 0.9 | 35% | 35% | 20% | 10% |
| 悬疑 | 3.5 | 1.3 | 30% | 30% | 25% | 15% |
| 武侠 | 3 | 1.4 | 20% | 25% | 30% | 25% |
| 古风 | 5 | 1.0 | 25% | 30% | 25% | 20% |
| 家庭伦理 | 5.5 | 0.95 | 20% | 35% | 30% | 15% |

### 镜头类型动态调整

根据故事位置自动调整镜头类型分布：

| 故事位置 | 调整规则 |
|---------|---------|
| 开头（position < 0.1） | 全景占比 +20%，特写占比 -10%（建立场景感） |
| 高潮区（0.4 < position < 0.7） | 特写占比 +10%（增强情绪张力） |
| 结尾（position > 0.8） | 全景占比 +15%（收束感） |

### 镜头时长动态调整

| 故事位置 | 时长倍率 | 说明 |
|---------|---------|------|
| 开头/结尾（<0.1 或 >0.9） | 1.2 | 放慢节奏，给观众时间进入/消化 |
| 高潮区（0.45-0.65） | 0.8 | 加快节奏，增强紧张感 |
| 其他区域 | 1.0 | 正常节奏 |

**最终时长** = 基础镜头时长 * 位置倍率 * (1 + 随机波动 * 0.4)，限制在 2-10 秒范围内。

### 运镜选择

| 故事位置 | 运镜偏好 |
|---------|---------|
| 开头（<0.1） | 推拉（50%）或 固定（50%） |
| 高潮区（0.45-0.65） | 随机选择（固定/推拉/平移/跟随/旋转 均等） |
| 结尾（>0.85） | 固定 |
| 其他区域 | 固定（40%）、推拉（25%）、平移（15%）、跟随（10%）、旋转（10%） |

### 情绪递进

每种题材有独立的情绪递进序列，根据故事位置（0-1）映射到对应情绪：

| 题材 | 情绪序列 |
|------|---------|
| 玄幻 | 平静 -> 惊讶 -> 紧张 -> 愤怒 -> 激昂 -> 释然 |
| 都市 | 平静 -> 好奇 -> 紧张 -> 期待 -> 喜悦 -> 满足 |
| 甜宠 | 平静 -> 羞涩 -> 甜蜜 -> 感动 -> 幸福 -> 温馨 |
| 悬疑 | 平静 -> 疑惑 -> 紧张 -> 恐惧 -> 震惊 -> 释然 |
| 武侠 | 平静 -> 警惕 -> 愤怒 -> 激昂 -> 豪迈 -> 释然 |
| 古风 | 平静 -> 羞涩 -> 担忧 -> 坚定 -> 感动 -> 温馨 |
| 家庭伦理 | 平静 -> 焦虑 -> 争吵 -> 理解 -> 感动 -> 温馨 |

## 角色圣经生成

### 生成规则

1. 从选题分析的 `character_proposals` 获取角色信息
2. 为每个角色生成视觉特征（面部、发型、服装、标志性特征）
3. 输出结构化的角色圣经

### 视觉特征模板

| 角色类型 | 面部 | 发型 | 服装 | 标志性特征 |

#### 都市题材

## 使用方法

### SOLO 执行步骤

1. 接收选题分析模块的输出
2. 根据题材类型获取节奏配置参数
3. 计算复杂度分数和最终镜头数
4. 按四幕结构分配镜头
5. 为每个镜头生成：
   - 场景描述（优先使用冲突点和关键场景）
   - 镜头类型（根据位置动态调整分布）
   - 运镜方式（根据位置偏好选择）
   - 时长（根据位置动态调整）
   - 情绪（根据情绪递进序列）
   - 台词（根据题材模板）
6. 生成角色圣经（视觉特征 + 性格）
7. 生成场景库（默认场景 + 关键场景）
8. 匹配音频风格
9. 输出完整的策划文档

## 反模式

| 错误做法 | 问题说明 | 正确做法 |
|---------|---------|---------|
| 使用固定镜头数不根据题材调整 | 玄幻需要更多镜头，甜宠可以更少 | 根据题材的节奏配置动态计算镜头数 |
| 忽略复杂度计算 | 简单故事和复杂故事用同样多的镜头 | 综合冲突点、关键场景、文本长度计算复杂度 |
| 镜头类型不随位置变化 | 全程中景，缺乏视觉节奏 | 开头多用全景建立场景，高潮区多用特写增强情绪 |
| 情绪不递进 | 全程同一情绪，缺乏叙事张力 | 按题材的情绪序列从平静到高潮再到释然 |
| 不生成角色圣经 | 后续素材生成缺乏一致性参考 | 必须为每个角色生成详细的视觉特征 |
| 场景库只有默认场景 | 缺少故事特有的场景 | 默认场景 + 从关键场景提取的特有场景 |
