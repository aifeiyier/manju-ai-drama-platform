## 核心功能

1. **选题研究** - 热点话题、关键词分析、竞品分析
2. **标题生成** - AI 直接生成多角度爆款标题
3. **正文创作** - AI 生成符合小红书调性的完整正文
4. **标签策略** - 5个标签黄金法则
5. **封面设计** - 视觉风格与文案建议
6. **发布时间优化** - 最佳发布时段分析
7. **数据分析** - 收藏率/点赞率/评论率核心指标解读
8. **博主诊断** - AI 分析账号阶段，给出个性化运营建议
9. **账号定位** - 昵称建议、简介框架、视觉风格
10. **内容规划** - 批量内容日历、系列内容策划
11. **变现路径** - 按粉丝量级给出变现建议
12. **竞品拆解** - 账号内容结构、爆款规律分析

## 平台核心指标

| 指标 | 健康值 | 优化方向 |
|------|--------|---------|
| 收藏率 | >3% | 提升实用感、增加干货密度 |
| 点赞率 | 3-8% | 优化封面标题 |
| 评论率 | 0.5-2% | 主动提问、设置讨论话题 |
| 分享率 | 0.5-1% | 增强实用性、收藏价值 |
| 封面点击率 | >5% | 优化封面视觉效果 |

> 收藏率是核心指标：收藏 = 用户觉得有用，是算法最看重的信号

## 标签策略

### 5个标签黄金法则

| 类型 | 数量 | 举例 |
|------|------|------|
| 流量大词 | 1个 | #穿搭 |
| 垂直领域词 | 2个 | #职场穿搭 #小个子穿搭 |
| 内容类型词 | 1个 | #穿搭分享 |
| 场景/人群词 | 1个 | #上班族 |

### 标签选择原则

- 选用户会搜的词，不要自创
- 宁精勿多：5个精准 > 10个模糊
- 大小搭配：大流量词+精准词

## 快速工作流

```
Step 1: 选题研究
  → 分析选题方向、关键词热度

Step 2: AI 生成多个标题
  → 生成3-5个不同角度标题，A/B测试

Step 3: AI 生成完整笔记正文
  → 符合小红书调性，每2-3句一个emoji

Step 4: 生成封面文案
  → 视觉风格与文案建议

Step 5: 博主诊断
  → 了解当前阶段该做什么

Step 6: 账号定位
  → 新人起步先用这个
```

## 思维模式：Utility-First, Collectibility-Driven

> 原则：收藏率第一，实用感至上，让用户觉得"这篇值得收藏"。

### 四步工作法

1. **对标与选题（Research & Topic）** - 先看3-5个对标账号，学习爆款规律
2. **小批量测试（Test Small）** - 先发3-5篇测试笔记，密切观察收藏率
3. **放大爆款（Scale Winners）** - 收藏率高的笔记，立即复制爆款逻辑
4. **复盘迭代（Review & Iterate）** - 每篇发布后24小时复盘

### 分阶段工作流程

- **Phase 1: 账号定位与对标** - 明确方向，找到对标，理解小红书生态
- **Phase 2: 小批量测试** - 找到感觉，验证方向
- **Phase 3: 稳定输出与放大** - 找到爆款逻辑后，稳定输出
- **Phase 4: 数据复盘与迭代** - 持续优化，越做越好
- **Phase 5: 变现与增长** - 粉丝过万后，开始变现

## 反模式清单

### 标题生成不要这样写

| 错误写法 | 问题 | 正确写法 |
|---------|------|---------|
| "最值得买的10款面霜" | 绝对化用词，违规 | "亲测！这10款面霜真的好用" |
| "用它皮肤真的变好了！" | 无依据、太泛 | "用了2周，下巴的痘消了一半" |
| "所有女生都要知道的事" | 绝对化 | "女生真的要试试这个方法" |
| "99%的人都不知道" | 夸大虚假 | "这个技巧用了3个月，真实有效" |

### 笔记正文不要这样写

| 错误写法 | 问题 | 正确写法 |
|---------|------|---------|
| "这款面霜含有丰富保湿成分" | 太广告、像说明书 | "涂上去润润的，第二天脸还是软软的" |
| "我推荐大家都去买" | 绝对化、违规 | "我自己已经回购了3瓶，真的喜欢" |
| "三天美白、七天淡斑" | 医疗宣称，违规 | "用了一周，感觉肤色亮了一点" |
| 大段文字不带emoji | 阅读疲劳 | 每2-3句一个节奏点，适当emoji |
| "适合所有肤质" | 无依据、违规 | "我是混干皮，用着OK，仅供参考" |

### 运营避坑清单

| 错误做法 | 问题说明 | 正确做法 |
|---------|---------|---------|
| 不看数据就随便发笔记 | 不知道什么内容受欢迎 | 每篇笔记后复盘收藏率/点赞率/评论率 |
| 标签堆10个以上 | 分散权重，反而没流量 | 严格遵守5个标签黄金法则 |
| 笔记没有emoji | 阅读体验差 | 每2-3句一个emoji，增加节奏感 |
| 内容太长不分段 | 用户看不下去 | 用小标题分割，每段不超过3行 |
| 所有笔记用一样的封面 | 用户审美疲劳 | 生成不同风格封面 |
| 不回复评论区 | 互动率低 | 评论区及时回复 |
| 刚起号就想接广告 | 粉丝信任度不够 | 先做优质内容积累粉丝 |
| 什么内容火就抄什么 | 没有自己的风格 | 借鉴爆款逻辑，加入独特视角 |
| 忽略收藏率只看点赞 | 收藏率才是核心指标 | 提升实用感和干货密度 |
| 发布时间不固定 | 流量不稳定 | 固定时间发布培养用户习惯 |

## 合规红线

1. 不用绝对化用语（"最"、"第一"、"99%的人不知道"、"保证有效"）
2. 不做医疗效果宣称（护肤品、保健品不得宣称治疗效果）
3. 不做虚假优惠（"今天最后一天"、"亏本甩卖"）
4. 不诱导私信（"私信领取"、"加微信发你"）
5. 不刷量买赞买收藏
6. 不碰敏感话题（政治、社会事件、擦边内容）
7. 不未经授权搬运他人内容
8. 不夸大效果（"三天变白"、"七天瘦十斤"）

### 合规表达对照

| 不要说 | 改为 |
|--------|------|
| 这个产品保证有效 | 我用下来觉得... |
| 所有人都适合 | 我个人感受是... |
| 强烈推荐大家都买 | 分享我的真实体验 |

## 约束与校准

### 必须做的（Hard Constraints）
- 每篇发布后24小时必须复盘，收藏率是核心指标
- 必须先发3-5篇测试笔记，找到感觉再放大
- 必须生成多个标题，A/B测试
- 必须严格遵守5个标签黄金法则（不多不少，刚好5个）
- 必须每2-3句加一个emoji，提升阅读体验
- 必须用小标题分段，每段不超过3行，留白舒适
- 粉丝过万前不要急着变现

### 推荐做的（Soft Constraints）
- 每周至少更新3篇，固定在最佳时段发布
- 收藏率高的笔记立即复制爆款逻辑
- 做内容日历，打造系列内容
- 评论区及时回复，提升互动率
- 定期分析对标账号，保持敏感度
- 结尾主动提问，引导评论互动
