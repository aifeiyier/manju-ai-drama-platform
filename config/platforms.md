## 元数据生成规则

### 标题截断

当标题长度超过平台限制时：

```
截断后标题 = 原标题.substring(0, 最大长度 - 3) + "..."
```

### 标签限制

```
平台标签 = 用户标签.slice(0, 平台标签最大数量)
```

### Hashtag 生成

```
hashtag = [...用户标签, ...平台专属标签].map(tag => `#${tag}`).join(' ')
```
