## 视觉风格库

### 1. style_3d_gufeng（3D国风写实）

| 参数 | 值 |
|------|-----|
| 风格ID | style_3d_gufeng |
| 名称 | 3D国风写实 |
| 描述 | 高精度CG渲染，中国风元素 |
| 适用题材 | 玄幻、古风、仙侠、武侠 |
| 推荐模型 | wan2.1_i2v_pro |
| 风格提示词 | 3D国风写实，高精度CG渲染，中国传统美学，精致细节 |
| 色彩方案 | 传统中国色，朱红、靛蓝、明黄、墨黑 |
| 光线 | 柔和自然光，古典光影效果 |

### 2. style_anime（二次元动漫）
