# Manju · AI漫剧创作平台

> AI 驱动的短视频/短剧全链路自动化创作工作台

---

## 🎬 项目简介

**Manju** 是一个面向短视频/短剧创作者的 AI 驱动创作平台。用户只需输入一个故事创意，平台即可自动完成从**选题分析 → 故事策划 → 素材生成 → 剪辑合成 → 导出成片**的全链路创作流程。

### 核心亮点

- 🎯 **5步创作向导** — 从故事创意到成片导出，全程引导
- 🧠 **8种题材智能识别** — 玄幻/都市/甜宠/悬疑/武侠/古风/家庭伦理/奇幻
- 🎨 **5种视觉风格 + 4种音频风格** — 智能匹配或手动选择
- 📱 **4大平台一键适配** — 抖音/快手/小红书/视频号参数自动校验
- 📐 **实时计算引擎** — 镜头数/时长/文件大小/质量检查全部实时计算

## 🛠️ 技术实现

### 构建工具

- **纯前端实现** — 单文件 HTML 应用（2000行），零外部依赖
- **Vanilla JavaScript** — 无框架，所有计算逻辑均为原生 JS 实现

### 技能体系

平台底层由 **13 个技能模块** 构成完整的创作知识库：

| 类别 | 技能 | 功能 |
|------|------|------|
| **核心创作** | AI短剧导演 | 4阶段工作流，需求澄清→锚点建立→分镜脚本→Prompt工程 |
| | 选题分析 | 8种题材自动识别，故事大纲/角色/冲突/受众分析 |
| | 故事策划 | 动态镜头计算，四幕结构分配，角色圣经生成 |
| | 素材生成 | 5种视觉风格+4种音频风格智能匹配引擎 |
| | 剪辑合成 | 10种转场效果，SRT字幕生成，4档质量预设 |
| | 导出成片 | 4平台适配，质量校验，元数据自动生成 |
| | 工作流引擎 | JSON工作流定义，变量引用，自动化串联 |
| **运营工具** | 抖音运营助手 v1.3 | 13项运营功能全链路覆盖 |
| | 小红书运营助手 v2.0 | 12项运营功能全链路覆盖 |
| **通用工具** | 浏览器自动化 | Rust高性能CLI，网页抓取/表单操作/截图 |
| | 智能数据分析 | Chat2DuckDB引擎，多格式数据查询 |
| | Agent团队创建 | 5种预设模板，自动构建Agent团队 |
| | 技能质量评价 | 8维评分体系，科学技能评估 |

### 创作流水线

```
故事创意 → 选题分析 → 故事策划 → 素材生成 → 剪辑合成 → 导出成片
  (输入)    (AI分析)   (算法计算)   (风格匹配)   (转场+字幕)   (平台适配)
```
