五个步骤，从创意到成片，AI 全程辅助

详细步骤说明

1

选题分析

输入你的故事创意描述（至少10个字），系统将自动分析题材类型、匹配度评分，并生成故事大纲、角色设定和冲突点建议。支持玄幻、都市、甜宠、悬疑、武侠、古风、家庭伦理七大类型。

2

故事策划

配置目标时长、视频模型和镜头参数。系统将根据故事复杂度自动计算分镜数量，按四幕结构分配镜头，生成视觉时间线和角色圣经。

3

素材生成

选择视觉风格和音频风格（可开启自动匹配），系统为每个镜头生成对应的视觉提示词和音频提示词，形成完整的素材清单。

4

剪辑合成

配置转场效果、画质预设和字幕选项。系统生成完整的时间线方案，包含镜头排列、转场标记、SRT 字幕预览和文件大小估算。

5

导出成片

选择目标平台，填写标题、描述和标签。系统自动进行质量检查，生成符合平台规范的导出方案和推荐话题标签。

💡 创作技巧

  • 故事描述越详细，选题分析越准确。建议包含时代背景、主要角色、核心冲突等要素。
  • 短剧建议控制在 60-90秒，中长剧可选择 180秒。
  • 不同视频模型适合不同风格：可灵擅长写实，即梦擅长国风，Runway 擅长特效，Pika 擅长动画。
  • 转场效果不宜过多使用花哨类型，淡入淡出和溶解过渡最通用。
  • 发布前务必进行质量检查，确保分辨率、码率、时长等参数符合平台要求。

❓ 常见问题

支持哪些题材类型？
目前支持七大类型：玄幻、都市、甜宠、悬疑、武侠、古风、家庭伦理。系统通过关键词匹配自动识别题材，并给出匹配置信度。
镜头数量是如何计算的？
基础镜头数 = max(8, 目标时长 / 单镜头时长)，然后根据故事复杂度（冲突点数量、关键场景数、文本长度）和题材系数进行调整。最终按四幕结构分配。
可以跳过某些步骤吗？
可以点击步骤指示器直接跳转到任意步骤。但建议按顺序完成，因为后续步骤依赖前序步骤的数据。跳过时系统会使用默认值。
导出的视频格式是什么？
导出为 MP4 格式（H.264 编码），分辨率和码率根据所选平台自动配置。草稿画质约 2Mbps，电影级可达 15Mbps。
如何选择合适的视觉风格？
可以开启"自动匹配"让系统根据题材类型推荐。一般来说：玄幻/古风选 3D国风写实 或 仙侠水墨，甜宠/校园选 二次元动漫，悬疑选 写实风格，家庭伦理选 卡通风格。

  • 未配置 API Key 时，所有功能使用本地模拟逻辑
  • 配置保存后刷新页面不会丢失
