# Manju · AI漫剧创作平台

## 1. 摘要

通过输入一个故事创意，平台自动完成选题分析、故事策划、素材生成、剪辑合成、导出成片的全链路创作流程。内置8种题材智能识别引擎，支持4大短视频平台一键适配，帮助创作者从灵感到成片全程提效，降低AI短剧创作门槛。

## 2. 背景

我是一名内容创作者，日常需要大量产出短视频和短剧内容。传统创作流程中，选题靠直觉、剧本靠手写、平台参数靠记忆，效率低下且容易出错。随着AI生成内容（AIGC）工具的普及，我发现虽然AI能生成视频和图片，但**从创意到成片之间缺少一个系统化的创作工作流**——创作者需要自己串联选题、策划、生成、剪辑、导出等环节，学习成本高、操作碎片化。

因此我决定开发一个**AI驱动的创作工作台**，将整个短剧创作流程整合到一个界面中，让创作者只需关注创意本身，其余交给系统自动完成。

## 3. 实践过程

### 项目概述

开发一个 AI 漫剧创作工作台，用户输入故事创意后，平台自动完成5步创作流程：选题分析 → 故事策划 → 素材生成 → 剪辑合成 → 导出成片。同时提供技能文档库浏览、风格配置查看、使用指南等功能。

技术栈：纯 HTML + CSS + Vanilla JavaScript

零依赖单文件应用，部署到 GitHub Pages

### 核心特性

- **5步创作向导**：从故事创意到成片导出，全程引导式操作，每一步都有实时计算和智能推荐
- **8种题材智能识别**：基于关键词匹配算法，自动识别玄幻/都市/甜宠/悬疑/武侠/古风/家庭伦理/奇幻题材，置信度评分
- **内容提取引擎**：从用户自然语言输入中自动提取角色、情节元素、核心概念，生成贴合用户创意的故事大纲
- **动态镜头计算**：基于四幕结构（起承转合），根据题材复杂度动态计算镜头数量和时长分配
- **5种视觉风格 + 4种音频风格**：智能匹配或手动选择，覆盖主流短剧审美
- **4大平台一键适配**：抖音/快手/小红书/视频号，自动校验分辨率、码率、时长、标题长度、标签数量
- **实时计算引擎**：镜头数、总时长、文件大小估算、质量检查全部实时计算，无需手动

## 5. 效果与总结

通过本项目，我实现了以下目标：

1. **创作流程标准化**：将碎片化的短剧创作流程整合为5步标准化工作流，创作者不再需要在多个工具之间来回切换
2. **AI能力平民化**：通过智能题材识别和内容提取算法，降低了AI创作工具的使用门槛，即使不懂Prompt工程也能产出专业级内容
3. **多平台一键适配**：自动处理不同平台的技术参数差异，创作者只需关注内容本身
4. **零依赖轻量化**：单文件HTML应用，无需安装、无需后端、无需数据库，打开浏览器即可使用

整个项目从需求分析到上线部署，完全使用AI编程助手完成，验证了AI在非研发场景（内容创作工具）中的实际应用价值。
